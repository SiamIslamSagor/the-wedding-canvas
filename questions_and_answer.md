###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
let count = 0;
const nums = [0, 1, 2, 3];

nums.forEach(num => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: C

<i>Here 0 is a negative number, zero followed by three positive numbers. The value of count is incremented by one for each positive number through the for each loop. The count starts from 0 and since three is a positive number, the value of the count is three thereafter. answer is C</i>

</p>
</details>
